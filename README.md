# jzqiu.com

🐬🐬🐬

## Develop React app:
  
Fresh clone:  
```npm install```  
  
Every time after that:  
```npm start```  

## Production build with Flask backend (no live edit on front end!)

Fresh clone:  
```npm install```  
```npm run build```  

Start backend server:  
```export FLASK_APP=app.py```  
```flask run```
